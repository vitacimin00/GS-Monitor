🔧 Setup Firebase
Cara mendapatkan Firebase Config:
1. Buka console.firebase.google.com
2. Klik "Add Project" → buat project baru
3. Di project, klik ⚙️ Project Settings
4. Scroll ke bawah, klik "Add App" → pilih Web (</>)
5. Copy firebaseConfig object-nya
6. Di menu kiri, klik "Realtime Database" → "Create Database"
7. Pilih region → Start in Test Mode → Enable
